¿Quién soy?

Soy estudiante de Ingeniería en Sistemas apasionado por la programación y el desarrollo de software. Desde mis primeros cursos descubrí el entusiasmo por construir soluciones que sean funcionales, prácticas y eficientes. Me gusta aprender constantemente nuevas tecnologías y enfrentar desafíos que pongan a prueba mis habilidades técnicas y creativas. Disfruto trabajar en equipo, compartir conocimientos y crear aplicaciones que resuelvan problemas reales.

Tecnologías que manejo:

  • 💻 HTML5
  • 🎨 CSS3
  • ⚙️ JavaScript
  • ☕ Java
  • 🛢️ MySQL
  • 🌐 Tailwind CSS
  • 🔧 Git
  • 🐙 GitHub

Otros intereses:

Me interesa la inteligencia artificial, la robótica, el diseño de interfaces, y el desarrollo de software educativo. También disfruto de la electrónica con Arduino y explorar nuevas herramientas que puedan hacer mis proyectos más innovadores.
